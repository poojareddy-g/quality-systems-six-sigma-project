# Quality Systems Process Improvement Project

Engineering Management quality systems project focused on reducing lipstick packaging defects during transportation using DMAIC methodology.

## Project Overview

This project analyzes a manufacturing quality issue involving lipstick melting during transportation due to insufficient heat resistance in packaging materials.

The objective was to reduce the rejection rate from 5% to 1%.

## Methodology

The project follows the DMAIC framework:

- Define
- Measure
- Analyze
- Improve
- Control
